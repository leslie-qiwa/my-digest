# Go 1.27 Release Candidate 1 Released

大家好，Go 开发者们，

我们刚刚发布了 go1.27rc1，这是 Go 1.27 的候选发布版本。

该版本从 release-branch.go1.27 分支的 go1.27rc1 标签处切出。

请使用新版本运行您的生产负载测试和单元测试。您帮助测试这些预发布版本对我们来说非常宝贵。

如遇到任何问题，请通过问题跟踪器报告：
https://go.dev/issue/new

如果您还没有选择加入 Go 遥测，请考虑加入。Go 遥测有助于验证此候选发布版本及未来的 Go 版本。

您可以通过运行以下命令来选择加入：

$ go telemetry on

如果您已经安装了 Go，试用 go1.27rc1 的简便方法是使用 go 命令：

$ go install golang.org/dl/go1.27rc1@latest
$ go1.27rc1 download

您可以从以下常规地址下载二进制和源代码分发包：
https://go.dev/dl/#go1.27rc1

要了解 Go 1.27 的变更内容，请阅读草案发布说明：
https://tip.golang.org/doc/go1.27

祝好，
Dmitri 和 Cherry 代表 Go 团队
