# The European Social Stack

# 一份公开宣言

## 欧洲社交技术栈

### 开放社交平台与私密通讯

社交媒体对社会和公民的影响清楚地表明了变革的必要性。欧洲拥有强大的社交企业生态系统，以及在设计和运营社交协议方面的深厚专业积累。我们正在此基础上，摆脱那些实行威权治理和编辑立场的大型垄断平台，携手建立一个多元且具有韧性的欧洲信息基础设施。我们将共同保护人民和机构免受这些平台的干预。更重要的是，我们将把社交媒体产生的数千个就业岗位和数十亿欧元的收入带回我们的大陆，让这些收入在我们自己蓬勃发展的商业生态系统中循环流动。

从这个角度来看，开放的欧洲平台和协议，如联邦宇宙（Mastodon、PeerTube 或 Mobilizon）、大气层（Eurosky、Flashes、Tangled 或 Web Tiles）以及私密通讯系统（如 Matrix 或 XMPP），都是欧洲互补的解决方案，拥有共同的目标。

以下是我们构建欧洲社交网络的方式：

## 共同原则

我们共同秉持的八项原则。

-
### 数字自主权

我们依据民主原则治理欧洲的数字基础设施，摆脱外国公司或政府的控制。

-
### 欧洲技术栈

社交媒体和社交网络是外国技术主导地位的重要组成部分，也是外国政治干预欧洲的关键工具。因此，我们正在将社交网络建设为独立欧洲的关键层。

-
### 去中心化网络

欧洲是一个由27个主权国家、200多种语言和方言、242个地区、10万多个城镇和4.5亿人口组成的联盟，团结在协作与协调对所有欧洲人最有利这一理念之下。去中心化社交网络契合欧洲模式——多样性和地方自治通过透明、开放的标准连接成一个稳健而有韧性的整体。作为欧洲人，我们正在抓住机遇，转向具有韧性的去中心化社交网络和协议，远离寡头拥有的封闭花园。

-
### 欧洲内容。欧洲社交网络。欧洲基础设施。

我们呼吁各国政府、市政当局、公共服务媒体和公民机构：

- 优先在开放的欧洲社交媒体上发布内容。寡头拥有的基础设施遵循寡头的编辑路线，而这些都不符合公众利益。
- 投资欧洲内容以及欧洲社交网络所需的内容生产基础设施。
- 资助支持欧洲内容和网络的去中心化韧性技术及基础设施的开发。

-
### 广泛多元的所有权

我们支持社交网络的多种所有权结构，从社区到私人拥有，从非营利到风投支持，从合作社到本地企业。我们呼吁我们的民选官员展现政治领导力，停止迎合那些不服务于任何欧洲利益的模式。

-
### 多元商业模式

所有权的多样性带来商业模式的多样性。我们的系统赋能服务器运营商和企业家选择最符合其需求的商业模式，无论是广告支持（包括在欧洲开发的新型隐私保护广告技术）、软件即服务（SaaS）、订阅制、众筹、公共资助，还是他们能想到的任何其他方式。

-
### 强化民主

欧洲正处于两条战线的混合冲突中；我们的选举和政治生活正遭受外国代理人的直接攻击，他们利用社交媒体操纵公众舆论并使政治议程偏向以削弱我们。我们正在部署内置编辑多元性和外国信息操纵与干预（FIMI）监测功能的系统，以保护我们的政治体制免受影响，并使我们的民主在攻击下保持韧性。

-
### 开源

欧洲社交技术栈是一个将欧洲开源软件战略以有意义的方式带入普通欧洲人生活的机会。开源服务器、基础设施和应用程序使欧洲人的社交网络更加透明、安全和可靠。我们正在采用开源和开放协议解决方案来替代专有的外国技术。

## 互补技术

三个互补的技术栈。

网络化技术形成的制度适合应对不同环境中不同的治理需求。将这三个技术栈组合在一起，可以涵盖广泛的领域：

-
### 私密通讯

在群组私密通讯方面，开放标准（如 Matrix 或 XMPP）非常适合即时通信或家人朋友之间的分享。多个欧盟成员国已成功采用这些协议，包括用于安全和国防领域。

-
### 联邦宇宙

联邦宇宙基于 ActivityPub 构建，从相互交流的紧密草根社区中成长起来，将社交平台或数字出版物的各个方面连接到一个统一的部署中。其代表包括 Mastodon、PeerTube 和 Mobilizon。

-
### 大气层

大气层运行 AT 协议，其基础建立在大规模公共社交媒体系统之上，因此提供了大型民主系统应有的那种解耦治理模式。其实际应用可见于 Eurosky、Flashes、Tangled 或较新的 Web Tiles。

## 联合开发

签署方计划共同建设的内容。

签署本宣言的开放欧洲平台和协议计划探索在重要的联合开发领域展开合作。这些领域包括：

-
### Eurobridge（欧洲桥）

支持和开发 Eurobridge——一座由欧洲拥有和运营的桥梁，连接欧洲联邦宇宙服务器与 Eurosky 及大气层（基于 Bridgy Fed 构建）。

-
### 分布式身份

开发一套跨网络及更广泛范围（例如欧洲数字身份钱包）可使用的分布式身份系统，并保护个人数据。

-
### 内容审核

根据《数字服务法》（DSA）和《数字市场法》（DMA）开发内容审核基础设施，以支持开发者和社区管理非法内容，如儿童性虐待材料（CSAM）。

-
### 多协议集成

多协议集成，例如在以社区或公共互动为主的社交应用中使用 Matrix 处理私信或动态，使开发者和用户以简单、无感的方式享受各方优势。

-
### 代表权与资金筹措合作

各协议、网络、平台及其所支持社区的代表相互支持各自面向机构和资助方的方案，在联合代表事务上展开合作，联合提交资金申请（在适当的情况下），或承认为互补解决方案提供双重资助的必要性（在分别提交资金申请的情况下）。

-
### 完整的社交生态系统

共同构建一个完整的社交生态系统。我们面对的是拥有数万亿估值和数十亿用户的在位者，唯一比他们更大的方式就是生态系统化。建立在共享基础设施和协议上的生态系统赋能小型开发者快速创建社交应用，并让用户在应用之间自由迁移。我们创造的网络效应由所有人共享。
