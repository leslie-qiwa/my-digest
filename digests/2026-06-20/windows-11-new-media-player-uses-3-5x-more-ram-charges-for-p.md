# Windows 11 New Media Player Uses 3.5x More RAM, Charges for Popular Video Codecs

# Windows 11 新版媒体播放器内存占用高出 3.5 倍，热门视频编解码器需付费使用

它取代了 Groove 音乐和经典的 Windows Media Player。

作者：Devesh Beri 2026年6月18日

微软为 Windows 11 推出的新版媒体播放器正遭受批评，因为测试发现它的内存占用远高于经典的 Windows Media Player。此外，新播放器还将部分热门编解码器置于付费墙之后。

据报道，新版媒体播放器在空闲状态下占用约 377MB 内存，而旧版播放器仅需约 103MB——在完全无操作的情况下，内存占用约为旧版的 3.5 倍。该程序打开本地视频文件所需的时间也更长，根据 Windows Latest 的测试，启动时间从旧版播放器的约两秒增加到新版的约三秒，增幅约为 50%。

争议不仅仅在于性能表现。微软现在将 HEVC（H.265）播放功能置于付费墙之后，需要通过 Microsoft Store 购买"HEVC 视频扩展"应用才能使用。微软还确认，Windows 11 24H2 版本移除了内置的 AC-3（杜比数字）编解码器，这意味着在这些系统上，新版媒体播放器无法原生播放 AC-3 音频轨道。

这款新软件在所有 Windows 11 电脑上取代了 Groove 音乐和经典的 Windows Media Player。微软仍将经典的 Windows Media Player 作为可选组件提供，但显然正在将新版媒体播放器作为大多数用户的默认选择来推广。

不介意使用第三方软件的用户可以考虑 VLC 等播放器，它们自带编解码器，无需依赖微软的付费附加组件。
