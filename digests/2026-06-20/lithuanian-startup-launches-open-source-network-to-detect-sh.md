# Lithuanian startup launches open-source network to detect Shahed-type drones

一家初创公司与活动人士共同推出了一套开源无人机探测系统，该系统可连接参与志愿者手机中的数据。这一消息来自"无人机雷达"（Drone Rada，Dronuradaras.lt）倡议的新闻稿。

经过验证的志愿者将一部闲置的安卓智能手机连接电源，放置在家中窗户附近。设备上运行的应用程序会持续分析环境声音，搜索沙赫德型无人机引擎特有的低频声学特征。

当检测到可疑声音时，嵌入式算法会对声学特征进行分析。如果附近多台设备记录到相同信号，系统便可确定目标的可能位置。

"如今，我们身边到处都是能够看到、听到并记录周围环境的设备。我们的目标是将这些传感器连接成一个共同的网络，为社会提供额外的安全保障层，增强国家的韧性，"该创意背后的Mainline初创公司创始人奥德里乌斯·祖尤斯表示。

该倡议由两部分组成：一个带有交互式公共地图的监测平台，以及一个供选择参与的志愿者使用的专用传感器应用程序。

该项目目前有一支由20名专家组成的团队，以及来自技术和安全领域的合作伙伴，包括立陶宛人工智能协会、波罗的海科技学院的专家，以及立陶宛步枪手联盟的成员。

组织者的目标是招募10,000名活跃用户，并将运营扩展至波罗的海三国和波兰。其目标是在波罗的海地区建立最大的公民声学无人机探测网络，拥有10,000名活跃参与者。

未来，"无人机雷达"网络预计将不仅接入智能手机，还将纳入居民使用的监控摄像头的音频。

"无人机雷达"团队还在与立陶宛主要移动网络运营商讨论潜在合作，考虑在其电信基站上安装传感器。

"我们的第一步是连接数百部手机，使我们能够创建一张实时地图，显示检测到与无人机相关声音的位置。随着设备数量的增加，我们将能够构建更准确的态势图。

"需要强调的是，用户隐私始终是我们最优先考虑的事项之一。我们的目标是用尽可能少的数据来识别无人机并确定其位置，"祖尤斯说。
