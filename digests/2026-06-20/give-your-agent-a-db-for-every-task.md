## 已上线和正在开发中的功能

**MCP 服务器**

为 Claude Code、Cursor、Windsurf、Codex、Gemini CLI、VS Code 等提供一等公民级别的工具接口。一条命令即可安装：`ghost mcp install`。

**Schema 自省**

`ghost schema` 以代理优化的格式展示你的数据库结构。表、视图、约束、索引，全部格式化呈现，让 Claude 无需猜测即可编写准确的查询。

**MCP 提示词**

可复用的引导包，代理可以按需调用。目前已发布八个，涵盖 pgvector 语义搜索、混合文本搜索、TimescaleDB 设置和 Postgres Schema 设计。

**只读模式**

`read_only` 配置选项可防止删除、暂停和 SQL 写入等破坏性操作。让代理访问你的数据库，无需担心意外数据丢失。

**直接进入 psql**

`ghost psql` 可打开一个连接到你数据库的交互式 psql 会话。不再需要复制连接字符串。

**无头登录**

`ghost login --headless` 使用 OAuth 设备流，适用于没有浏览器的环境。沙箱、虚拟机、容器、远程服务器。在任何设备上输入验证码即可登录。

**内置 pgvector + pg_textsearch**

无需安装扩展。向量密集型数据库的分叉保持一致。代理从第一天起即可使用搜索功能。

**定时删除**

在 N 分钟后自动删除数据库。为那些忘记自己善后的代理而设。
