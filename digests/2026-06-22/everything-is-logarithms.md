# Everything is logarithms

# 万物皆是对数

一些事物之间的联系，我在别处没有见过。也许它们意味着什么？

## 1. 无底数的对数

通常我们书写带底数的对数 \(\log_b (x)\)，意为

\[y = \log_b (x) \Lra b^y = x\]

然后可以通过以下公式进行底数转换：

\[\log_b (x) = \frac{\log_a (x)}{\log_a(b)}\]

这由整理 \(\log_a (x) = \log_a (b^{\log_b x}) = \log_b (x) \times \log_a (b)\) 得出。

理解这个公式的一种方式是将其视为单位换算，类似于写 \(2 \text{ 千米} = 2000 \text{ 米} / \frac{1000 \text{ 米}}{1 \text{ 千米}}\) 或 \(5 \text{ 字节} = 40 \text{ 比特}/\frac{8 \text{ 比特}}{1\text{ 字节}}\)。它在问：\(x\) 中包含多少个 \(b\) 的副本？答案是 \(x\) 中包含的 \(a\) 的副本数，除以 \(b\) 中包含的 \(a\) 的副本数。

这完全是简单的，但不知何故用这种方式思考对数却很困难。符号系统在某种程度上……混淆了事物？具体来说，很难将 \(\log_b x\) 读作"x 中包含多少个 b 的副本"，因为这个中文表述应该对应符号 \(x/b\)，而非 \(\log_b x\)。"x 中包含多少个 b 的因子"好一些，但仍然感觉不太对。

我找到了一种思考对数的方式，我认为它更加清晰，但你需要允许一种我称之为**无底数对数**的奇特对象。它就是一个没有底数的对数：

\[\log N\]

我们将其视为一个抽象对象，而非一个数字。然后我们将通常的"有底数"对数写成两个无底数对数的比值：

\[\log_2 N = \frac{\log N}{\log 2}\]

注意，这已经是人们在口语中会做的事情，例如在渐近公式中省略对数的底数。但我的意思不是将其作为简写；将其视为一个实际的代数对象更为有用。

我们将 \(\log 2\) 解释为单位"比特"。将 \(\log N\) 用比特表示，就是将其分解为 \(\log 2\) 的倍数：

\[\log N = \frac{\log N}{\log 2} \log 2 = \log_2 (N) \log 2 = \log_2 (N) \text{ 比特}\]

然后对数的换底公式就来自于用不同的单位书写同一个几何量。例如 \(\log e\) 作为单位有时被称为"奈特"：

\[\begin{aligned} \log N = \frac{\log N}{\log 2} \log 2 = \log_2 (N) \text{ 比特} = \frac{\log N}{\log e} \log e = \ln (N) \text{ 奈特} \end{aligned}\]

无底数的 \(\log N\) 有点像在向量讨论中可能熟悉的一个对象的乘法版本。在向量中，区分点和位移是很常见的：位移向量 \(\b{v}\) 由两个点的差给出 \(\v = (b) - (a)\)。当我们将点视为具有坐标时，这涉及对原点 \(\O\) 的显式选择，使得 \(\b{a} \equiv (a) - \O\) 和 \(\b{b} \equiv (b) - \O\)。然后位移向量通过减去 \(\O\) 的因子来构建，\(\b{v} = \b{b} - \b{a} = ((b) - \O) - ((a) - \O) = (b) - (a)\)。无底数对数实现了同样的事情，只是用乘法代替：值 \(\log N\) 可以被视为对于未指定的原点选择的 \(\log N / \log \O\)；将其转换为实际数值涉及两个这样的对数相除以消去原点，\(\log_M N = \log N / \log M = (\log N / \log \O) / (\log M / \log O)\)。我将 \(\log N\) 视为对应于 \(N\) 的点，将 \(\log N / \log \O\) 视为选择坐标系后的相应位移向量。点的版本更为基本。

你可能会问：如果我们有无底数对数 \(\log N\)，是否也有"无底数指数"？通常 \(b^{\log_b N}\) 可以写成类似 \(b^{\log_b N} = b^{\ln N / \ln b} = e^{\ln N} = N\) 的形式；有没有办法在不实际选择底数的情况下做到这一点，比如 \((\ast)^{\log N}\) 之类的？我认为答案必须是"没有"，因为我想不出让它有意义的方法。我们只能说，我们已经将一个对象——对数 \(\log_b N\)（即 \(b^y = N\) 的解）——拆分成两个对象 \(\log N\) 和 \(\log b\)，它们各自单独都没有"单位"，因此没有数值意义。

所以对数的行为有点像乘法向量，在这个意义上它们必须相对于一个"原点"即底数的选择来定义。事实上，对数和向量之间有许多令人惊讶的相似之处，我很享受地阐述了这些：

## 2. 对数即是向量

在以适当协变的方式进行向量代数和微分几何时，我们区分抽象向量和特定坐标系中的向量。

我个人的约定是将抽象向量称为"几何"向量，并始终用粗体书写 \(\v\)，而"坐标"向量——即其在坐标中的值的元组——用箭头标记 \(\vec{v} = (v_x, v_y, v_z)\)。粗体几何向量始终与坐标无关，而坐标向量只是数字或其他对象的集合。几何向量 \(\b{v}\) 可以写成其坐标与基向量"标架" \(X = (\x, \y, \z)\) 的点积：

\[\b{v} = \vec{v} \cdot X = (v_x, v_y, v_z) \cdot (\x, \y, \z) = v_x \x + v_y \y + v_z \z\]

\(\v\) 在基向量 \(\x\) 上的投影通过用基向量"测量"该向量得到（基向量不必具有单位长度）。我喜欢将其写成除法，因为它的行为很像除法（尽管技术上是伪除法）：

\[\frac{\v}{\x} = v_x\]

这里我使用了自己非常非标准的向量除法记号。更常见的写法是用偏导数投影微分 \(df = f_x dx + f_y dy + f_z dz\) 的分量，这也是伪除法运算（顺便说一下，这就是偏导数在某种意义上像除法但又不完全是除法的原因）：

\[\frac{\p f}{\p x} = f_x\]

我会用两种形式书写以便于在它们之间转换；我确实更喜欢我的向量除法版本，因为它避免了引入微积分的无关记号，但由于后者确实是标准的，我也应该包含它以作比较。

假设 \(\b{v}\) 是一维的，\(\b{v} = v_x \x\)。那么在"量尺" \(\b{m} = m \x\) 上的投影以 \(m\) 的倍数来衡量其长度：

\[\frac{\v}{\b{m}} = \frac{v_x \x}{m \x} = \frac{v_x}{m}\]

再乘以 \(\b{m}\) 就是我们所说的"以 \(\b{m}\) 为单位书写 \(\b{v}\)"：

\[\frac{\b{v}}{\b{m}} \b{m} = (\frac{v_x}{m}) (m \x)\]

这里 \(m\) 是单位"米"，\(v_x/m\) 是 \(v_x\) 用米表示的值。当然，要实际计算 \(v_x/m\)，你首先需要有单位——但显然这与对数的情况是同类事物，你可以将 \(\b{v}\) 和 \(\b{m}\) 视为"无单位"的概念进行几何比较，然后 \(v_x/m\) 是它们在任意坐标系中的投影。

无底数对数对对数执行的是同样的操作，其中 \(\log N\) 扮演几何向量 \(\v\) 的角色，\(\log 2 = \text{比特}\) 是单位向量或量尺，扮演 \(\x\) 的角色。

\[\begin{aligned} \frac{\log N}{\log 2} &= \log_2 N \\ \frac{\log N}{\log 2} \log 2 &= \log_2 N \text{ 比特} \end{aligned}\]

在这个意义上，无底数对数以完全相同的方式用坐标书写数字，就像量尺用坐标书写向量一样。

不同单位下对数的等价性

\[\begin{aligned} \log N &= \frac{\log N}{\log 2} \log 2 = \log_2 (N) \text{ 比特} \\ &= \frac{\log N}{\log e} \log e = \ln (N) \text{ 奈特} \end{aligned}\]

与不同单位下几何向量的等价性相同

\[\begin{aligned} \v &= \frac{\v}{\x} \x = v_x \x \\[1em] &= \frac{\v}{\x'} \x' = v_{\x'} \x' \\ \end{aligned}\]
