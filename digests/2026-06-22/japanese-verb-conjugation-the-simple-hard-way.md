# Japanese verb conjugation the simple hard way

日语动词太棒了。你可以拿一个动词的辞书形（比如"taberu"，吃），然后变换出许多形式：

taberu（"吃"）：tabemasu（礼貌体"吃"）、tabemashita（礼貌体"吃了"）、tabemasen（礼貌体"不吃"）、tabemasendeshita（礼貌体"没吃"）、tabenai（随意体"不吃"）、tabenakatta（随意体"没吃"）、tabetai（"想吃"）、tabeta（随意体"吃了"）、tabete（"吃了然后……"）。

nomu（"喝"）：nomimasu（礼貌体"喝"）、nomimashita（礼貌体"喝了"）、nomimasen（礼貌体"不喝"）、nomimasendeshita（礼貌体"没喝"）、nomanai（随意体"不喝"）、nomanakatta（随意体"没喝"）、nomitai（"想喝"）、nonda（随意体"喝了"）、nonde（"喝了然后……"）。

正确制作这些形式的技术（或科学）叫做动词变位。我之前曾多次尝试学习日语动词变位。乍一看很简单（不就是换个后缀嘛！），但作为学习者，有很多微妙之处会把你拖入泥潭。

我找到了一套我更喜欢的体系，但先让我解释一下我为什么曾经学得那么痛苦。

## 动词变位通常是怎么教的

一种常见的变位教学方式大概是这样的：

1. 动词分为两组（通常称为-ru动词和-u动词）。
2. 对于每个后缀（比如-masu），你需要学习如何正确地将它附加到两组动词上（使用类似这样的表格）。

我觉得这种教学方式让人深感沮丧和不满。

首先，把它们叫做-ru动词和-u动词很容易混淆，因为你不能总是通过词尾来判断分组。有时候符合你的预期（"taberu"，吃，是-ru动词；"nomu"，喝，是-u动词）。好吧，那如果一个词以-ru结尾，它肯定就是-ru动词吧？其实未必！"kaeru"（回家）是-u动词。

好吧，真坑。

如果光是分组还不够让人困惑的话，再来看这个：你还得记住十来条看似随意的规则，比如"く（ku）在masu前变成き（ki）"，针对不同的后缀各有不同。一些教材还会把-u动词进一步分成五个子类来教这些规则。

这让日语动词变位看起来很难！

然而，在这些表面的困难之下，隐藏着一个出人意料地优雅的体系——只是被常见的教学方式所遮蔽了。在这篇文章中，我将用一种"简洁但有深度"的方式来解释它。这意味着，我们不去死记硬背一堆个别的特殊情况，而是尝试从少量的基本要素出发，慢慢构建一个连贯的体系。

## 其实就是拼接

让我们回到这两个例子：

taberu（"吃"）：tabemasu（礼貌体"吃"）、tabemashita（礼貌体"吃了"）、tabemasen（礼貌体"不吃"）、tabemasendeshita（礼貌体"没吃"）、tabenai（随意体"不吃"）、tabenakatta（随意体"没吃"）、tabetai（"想吃"）、tabeta（随意体"吃了"）、tabete（"吃了然后……"）。

nomu（"喝"）：nomimasu（礼貌体"喝"）、nomimashita（礼貌体"喝了"）、nomimasen（礼貌体"不喝"）、nomimasendeshita（礼貌体"没喝"）、nomanai（随意体"不喝"）、nomanakatta（随意体"没喝"）、nomitai（"想喝"）、nonda（随意体"喝了"）、nonde（"喝了然后……"）。

你能在这两者之间发现什么规律吗？

如果你关注第一个例子，变位看起来很直观。没错，后缀很多，但每个都是通过纯粹的拼接来添加的：

tabe + ru = taberu
tabe + masu = tabemasu
tabe + mashita = tabemashita
tabe + masendeshita = tabemasendeshita
tabe + masen = tabemasen
tabe + nai = tabenai
tabe + nakatta = tabenakatta
tabe + tai = tabetai
tabe + ta = tabeta
tabe + te = tabete

我们把左边的部分——在这些形式之间保持不变的部分——叫做动词的**词干**。taberu的词干是"tabe"。

所以现在你（几乎）知道怎么做变位了：

> 要给日语动词变位，把后缀附加到词干上即可。例如，tabe + masu = tabemasu。

从某种意义上说，这差不多就是全部原理了！但我们现在需要加深对"词干"和"附加"这两个概念的理解。

## 通配元音

在第一个例子中，词干非常简单：就是tabe。

现在让我们仔细看看第二个例子：

nomu（"喝"）：nomimasu（礼貌体"喝"）、nomimashita（礼貌体"喝了"）、nomimasen（礼貌体"不喝"）、nomimasendeshita（礼貌体"没喝"）、nomanai（随意体"不喝"）、nomanakatta（随意体"没喝"）、nomitai（"想喝"）、nonda（随意体"喝了"）、nonde（"喝了然后……"）。

你能发现什么规律吗？

我想到了几点：

1. nomu的词干到底是什么并不完全清楚。是nomu吗？但我们把词干定义为不变的部分——而最后一个元音似乎在交替变化，有时是nomi，有时是noma。
2. 即使我们选nom作为词干（这很怪！你甚至不能用平假名写出"nom"，因为没有单独的"m"），这也不完全行得通，因为nonda和nonde打破了这个规律。

暂时先忽略nonda/nonde，让我们聚焦于第一个问题。

我们说nomu确实有一个词干。

我们把它写成这样：**nom\***。

等等，这是什么？

这里的\*就像一个**通配元音**，"等待被填充"。它可能是noma、nomi、nomu、nome或nomo，取决于语境。在书写词干时，我总是中性地写成nom\*。

但实际的元音是如何决定的呢？让我们试着找找规律：

nom\* + masu = nomimasu
nom\* + mashita = nomimashita
nom\* + masendeshita = nomimasendeshita
nom\* + masen = nomimasen
nom\* + nai = nomanai
nom\* + nakatta = nomanakatta
nom\* + tai = nomitai

看起来每个后缀都"夹带"了一个通常不可见的首选秘密元音。我们想加"masu"，但得到的是"imasu"。我们想加"nai"，但得到的是"anai"，以此类推。

事实确实如此！

每个后缀都以一个（秘密元音）开头。只有当前面有\*通配符时它才会显现；否则就消失了：

nom\* + (i)masu = nomimasu
nom\* + (i)mashita = nomimashita
nom\* + (i)masendeshita = nomimasendeshita
nom\* + (i)masen = nomimasen
nom\* + (a)nai = nomanai
nom\* + (a)nakatta = nomanakatta
nom\* + (i)tai = nomitai

对于tabe来说，没有通配符，所以后缀的秘密元音无处容身。在这些情况下，它直接被丢弃：

tabe + (i)masu = tabemasu
tabe + (i)mashita = tabemashita
tabe + (i)masendeshita = tabemasendeshita
tabe + (i)masen = tabemasen
tabe + (a)nai = tabenai
tabe + (a)nakatta = tabenakatta
tabe + (i)tai = tabetai

因此，我们似乎可以制定一条规则来完善拼接法：

> 后缀可能以一个秘密元音开头。如果词干以通配符结尾，元音就会显现：nom\* + (i)masu = nomimasu。否则，元音被丢弃：tabe + (i)masu = tabemasu。

有了这个改进，我们现在可以正确地变位绝大多数动词了。让我们暂停一下，看看我们目前到了哪里。

## 原来这就是动词分组

到目前为止，我们已经识别出两类动词：

tabe + (i)masu = tabemasu
tabe + (i)mashita = tabemashita
tabe + (i)masendeshita = tabemasendeshita
tabe + (i)masen = tabemasen
tabe + (a)nai = tabenai

以及

nom\* + (i)masu = nomimasu
nom\* + (i)mashita = nomimashita
nom\* + (i)masendeshita = nomimasendeshita
nom\* + (i)masen = nomimasen
nom\* + (a)nai = nomanai
nom\* + (a)nakatta = nomanakatta
nom\* + (i)tai = nomitai

现在让我们把它与你之前听说过的概念联系起来。

像taberu这样的动词叫做**-ru动词**，传统上称为"**一段**"动词（字面意思是"一行"）。而像nomu这样的动词则叫做**-u动词**，传统上称为"**五段**"。

为什么这么叫呢？

在一段动词（如taberu）中，词干的最后一个音节是固定的。无论后缀是什么，它永远是be：

它始终停留在平假名表的同一行上，因此叫"一段"。

另一方面，在五段动词（如nomu）中，词干的最后一个音节在ma、mi、mu、me和mo之间交替：

它横跨所有五行，这就是为什么叫五段。m\*这个"通配符"代表了整个ma/mi/mu/me/mo列。
