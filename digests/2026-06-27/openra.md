# OpenRA

测试版 20260222
过去一年带来了若干令人振奋的新进展，我们很高兴在今天通过全新的 OpenRA 测试版与大家分享！
playtest-20260222 中最重磅的新功能是为《红色警戒》《泰伯利亚黎明》和《沙丘 2000》推出的全新随机地图生成器。它们的使用方式正如你所预期的那样：选择一种生态群系、玩家数量，以及关于对称性和资源的一些细节，然后即可开战！生成的地图在遭遇战和多人对战中均可使用。
《沙丘 2000》迎来了一次焕新，为音速坦克和受损建筑加入了全新的视觉效果，并带来了星港期盼已久的“批量采购”逻辑。本次更新包含一套由社区主导、面向遭遇战和多人对战模式的完整平衡性大改。与此同时，单人战役也获得了自己的难度调整，以缓和上手难度。
在（目前）独立的《泰伯利亚黎明》高清模组中，对《命令与征服：重制版合集》素材在《泰伯利亚黎明》中的支持达成了两个重要的里程碑。该模组现已功能完备，为最后一批自定义素材加入了全新的高清精灵图，还配备了一个全新的内容管理器，让你可以在重制版与经典版的美术、音效和音乐之间进行选择。在将这些功能整合进核心 OpenRA《泰伯利亚黎明》方面也取得了重大进展——虽然本次发布它仍是一个独立模组，但我们希望在下一个版本中完成合并。
对于我们的地图制作社区而言，这又是一次重大发布，OpenRA 地图编辑器获得了进一步的界面改进，并新增了利用随机地图生成器逻辑的工具。
其他值得注意的变更包括：
- 新增了“其他 RTS”鼠标输入模式。
- 为任务/遭遇战新增了定时自动存档设置。
- 电脑 AI 现在会尝试建造扩张基地。
- 在支持未来版本本地化方面取得进一步进展。
- 为《红色警戒》和《泰伯利亚黎明》各新增了一个任务。
- 许多其他错误修复和小幅性能优化。
一如既往，完整的更新日志已经提供，包含关于各项变更和修复的更多信息。请持续关注更多更新，并务必参与本次测试。别忘了在我们的论坛、社区 Discord 服务器或 GitHub 上与我们分享你的反馈！
