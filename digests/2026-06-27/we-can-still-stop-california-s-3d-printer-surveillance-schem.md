# We can still stop California's 3D printer surveillance scheme

加州州议会无视EFF关于实施3D打印监控软件新强制规定的危险性和不可行性的警告，批准了正是要做这件事的立法。在此过程中，立法者修订了该法案，使其变得更加令人困惑，同时未能解决对隐私、言论和消费者权利的风险。我们必须重申呼吁立法者在该法案提交州参议院之际撤销它，并保护该州创作者的工具。

请加州参议员与创作者站在一起

该法案有什么变化？

自我们首次撰文讨论AB 2047以来——这项法案针对3D打印机，旨在打击无证制造枪支这一罕见、不切实际且早已被取缔的行为——它已增加了多项修正案。其中一些是受欢迎的改变，但大多数只是凸显了所提议方案在技术官僚层面的荒谬。我们的核心担忧——即这项强制规定审查合法言论、构建企业监控、并将开源实验定为犯罪——并未得到补救。

取消对转售的刑事定罪

先从一个好消息说起，目前的法案包含了对设备私人转售的豁免条款。原始法案会将个人转售在这项强制审查和监控软件规定之前购买的3D打印机定为刑事犯罪。这对3D打印社区来说是一个明显的胜利，但遗憾的是这还不够。

对开源的豁免条款无效

该法案最危险的方面之一是，它将用户的常见做法定为犯罪，比如为他们的3D打印机创建和使用替代性的开源程序。新修正案为开源工具的使用提供了豁免，但前提是它必须包含合规的审查软件。该法案为开源开发者强加了关于打印阻断的模糊且不切实际的标准，并继续对开源用户产生寒蝉效应。

取消任何实际有效的要求

重申一遍——在任何情况下，这项强制要求的技术都不可能按预期发挥作用。它既会阻止3D打印机的合法使用，又会允许任何铁了心要打印枪支的人打印枪支。没有任何修正案能改变这一现实。

相反，目前的法案干脆放弃了这项强制规定预期能起作用的伪装。算法的性能标准从"有效防止技术娴熟的用户规避[该算法]"改为"大幅降低可预见的规避企图的可能性……"。该法案仍将要求对所有打印进行监控，但它不再针对娴熟用户测试有效性，而只是与任何用户都能采用的（字面意义上）无限多种规避方式玩打地鼠游戏。

此外，该法案现在留给我们的是一个不明确的流程，它依赖非政府第三方来定义标准，并依赖制造商和经销商进行自我监管。

好莱坞分一杯羹

该法案还包含了又一项针对商业用户的豁免条款。这次是针对娱乐行业，该行业大量使用3D打印机制作道具和服装。

这对大型制片厂来说没问题，但它把独立电影制作人、cosplay爱好者和许多其他小型创作者排除在外。

这只是一项为限制企业反对而做的防御性修订。在3D打印领域，消费级工具和商业级工具之间并没有明确的界限。这些都是通用工具，可能被大型制片厂的道具部门使用，也可能被一位准备参加漫展的艺术家使用。事实上，消费级产品不仅被业余艺术家和提升技能的工程师使用。商业3D打印机，就像它们传统的2D对应物一样，经常在工作场所使用，也被磨练技能的专业人士使用，或者只是被想在家完成一些工作的人使用。

商业豁免条款让打印机制造商有能力销售更昂贵档次的打印机，从而锁定并向其商业客户多收费。其中一些客户会选择购买普通零售版本，但这会带来自身的代价：知识产权被盗的风险增加，因为所有打印文件都会像爱好者的文件一样被监控。这意味着企业面临着真实的风险，可能将任何原型或新设计不仅泄露给打印机制造商，还可能通过数据泄露泄露给窥探的政府和/或公众。

要求你的参议员反对AB 2047

这个更新版本的AB 2047降低了性能标准并取消了监督，同时仍然威胁着3D打印机用户的隐私和选择权。打印机监控系统无法实现其预期目的，只会伤害守法用户。
